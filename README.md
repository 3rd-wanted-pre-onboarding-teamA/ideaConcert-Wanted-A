## 프로젝트 목적
1. AWS EC2 환경에 간단한 nodejs 웹 서버를 구축
2. Nginx를 WAS 앞단에 두어 라우터 역할하도록 구성

<br/>
<br/>

## 상세 조건
1. Web Application Server : NGINX 사용 ( 80번 포트 오픈 )
2. Elastic IP를 통한 IP
3. IP/api/hello 로 접속하였을때 웹브라우저 상에서 Hello 표시
